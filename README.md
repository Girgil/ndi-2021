# Activiation du venv
## Si dossier venv pas encore créé :
```bash
python3 -m venv venv
```
(ou python si votre python3 est python)

## Activation du venv
### Linux:
```bash
source venv/bin/activate
```
### Windows:
```cmd
cd venv/Scripts
activate.bat
cd ../..
```

# Installation des dependances :
```bash
pip3 install -r requirements.txt
```
